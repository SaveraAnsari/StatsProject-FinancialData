##  Project Overview
This project implements and compares multiple regression techniques to predict **stock closing prices** using historical financial data.  
The goal is to evaluate **accuracy, stability, and generalization** across analytical, iterative, and regularized models.

Implemented methods include:
- **Ordinary Least Squares (OLS)**
- **Singular Value Decomposition (SVD)**
- **Gradient Descent (GD)**
- **Principal Component Analysis (PCA) Regression**
- **Ridge Regression**
- **Bonus Tasks:** QR Decomposition, PCA Cross-Validation, and Neural Network Regression
